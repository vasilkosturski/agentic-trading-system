# CI cycle reports

Auto-generated by the **Agentic staging check** workflow
(`.github/workflows/agentic-staging-check.yml`). Each run has headless Claude
verify the latest staging trading cycle, diff it against the previous report to
detect drift, and commit a new `CI_CYCLE_REPORT_<timestamp>.md` here.

- **Retention:** newest 30 markdown reports (older are pruned each run).
- **Content:** trading cycle + drift data only — no infra/security narrative
  (this is a public repo). See `.github/agentic-staging-check.md` for the rules.
- **Screenshots** are uploaded as CI artifacts, never committed here.

This differs from `reports/trading-cycles/` (gitignored local operational notes).
